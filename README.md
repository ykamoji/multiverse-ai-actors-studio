# Multiverse AI Actors Studio

Go to [Live App](https://ykamoji.github.io/multiverse-ai-actors-studio/)

## Overview

Multiverse AI Actors Studio is a tool that lets you create composite scenes with consistent characters using cutting-edge AI. By defining character portraits, descriptions, and poses, along with an overarching scene description, you can synthesize new scenes easily.

*Note: This is meant to be a low-tier Flow / Whisk alternative.*


## Key Features

*   **Virtual Cast Management**: Build and maintain a roster of AI "actors." Name them, describe their visual attributes, and upload reference portraits to anchor their identities.
*   **Scene Direction**: Write detailed scene descriptions, placing your characters in dynamic environments and situations.
*   **Action & Posing**: Assign specific poses, expressions, or actions to individual characters to control exactly how they behave and interact in the final composition.
*   **Consistent Synthesis**: Provide multiple distinct characters into a single prompt, and the AI will composite them together into a unified frame while doing its best to preserve their reference identities.
*   **Visual Storyboarding**: An intuitive visual interface ideal for writers, game designers, or comic artists wanting to rapidly prototype concepts and map out visual narratives without hand-drawing every panel.
